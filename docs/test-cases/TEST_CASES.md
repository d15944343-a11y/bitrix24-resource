# Тест-кейсы

## TC-001. Успешный вход администратора

### Предусловия

- приложение запущено
- в базе есть пользователь `admin@example.com`

### Шаги

1. Открыть страницу входа `/login`
2. Ввести email `admin@example.com`
3. Ввести пароль `admin123`
4. Нажать кнопку входа

### Ожидаемый результат

- пользователь успешно авторизован
- происходит переход в личный кабинет

## TC-002. Ошибка входа при некорректном email

### Предусловия

- приложение запущено

### Шаги

1. Открыть страницу входа `/login`
2. Ввести email `invalid-email`
3. Ввести любой пароль
4. Нажать кнопку входа

### Ожидаемый результат

- пользователь остается на странице входа
- отображается сообщение о некорректном email

## TC-003. Создание нового клиента

### Предусловия

- пользователь авторизован

### Шаги

1. Перейти в раздел клиентов
2. Открыть форму добавления клиента
3. Заполнить ФИО, email, телефон, город и статус
4. Сохранить клиента

### Ожидаемый результат

- клиент сохранен в базе
- открывается карточка созданного клиента

## TC-004. Запрет создания клиента с дублирующимся email

### Предусловия

- в базе уже существует клиент с email `ivan.petrov@example.com`
- пользователь авторизован

### Шаги

1. Открыть форму создания клиента
2. Ввести email `ivan.petrov@example.com`
3. Заполнить остальные поля
4. Нажать кнопку сохранения

### Ожидаемый результат

- клиент не создается
- отображается сообщение об ошибке

## TC-005. Ограничение доступа к админ-панели для менеджера

### Предусловия

### Шаги

1. Войти под менеджером

### Ожидаемый результат

## TC-006. Отправка формы обратной связи

### Предусловия

### Шаги

### Ожидаемый результат

## TC-007. Сохранение настроек интеграции Bitrix24

### Предусловия

### Шаги

1. Открыть страницу `/integration`

### Ожидаемый результат

## TC-008. Экспорт отчета по клиентам в CSV

### Предусловия

- в базе есть клиенты
- пользователь авторизован

### Шаги

1. Открыть страницу `/reports`
2. Нажать кнопку скачивания CSV-отчета

### Ожидаемый результат

- начинается загрузка файла `clients_report.csv`
- файл содержит список клиентов и их основные данные
