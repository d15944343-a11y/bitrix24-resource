# Bitrix24 Resource

Учебный дипломный проект на Flask по теме:

`Разработка информационного ресурса для интеллектуального анализа клиентской базы на основе CRM системы Битрикс 24`.

## Текущий статус

Реализован минимальный стартовый каркас проекта:
- структура Flask-приложения;
- отдельный конфигурационный модуль;
- общий базовый HTML-шаблон для последующего расширения;
- одна базовая страница;
- подключение статических файлов;
- точка входа для локального запуска.

## Запуск

1. Создать и активировать виртуальное окружение.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Запустить приложение:

```bash
python app.py
```

4. Открыть в браузере:

```text
http://127.0.0.1:5000/
```

## План развития

Проект будет развиваться поэтапно небольшими коммитами: конфигурация, шаблоны, авторизация, роли, SQLite, интеграция Bitrix24, аналитика, отчеты, тесты и документация.
