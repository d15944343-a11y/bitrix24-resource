# Bitrix24 AI Resource

Дипломный проект на Flask по теме:

`Разработка информационного ресурса для интеллектуального анализа клиентской базы на основе CRM-системы Bitrix24`.

## Что реализовано

Проект переделан из обычного веб-ресурса в демонстрационный интеллектуальный сервис:

- загрузка датасетов для обучения из локального CSV или по внешней ссылке;
- хранение реестра датасетов, обученных моделей и запусков прогнозирования;
- обучение ML-модели прогноза оттока клиентов;
- сохранение и повторная загрузка модели из `model.joblib`;
- применение модели к локальной CRM-базе клиентов;
- AI-аналитика, сегментация и рекомендации по удержанию;
- интеграция с Bitrix24 для импорта клиентов.

## Структура обязательных артефактов

- `main.py` — основной файл запуска приложения;
- `model.joblib` — сохранённая обученная модель;
- `data/crm_customer_churn.csv` — датасет для обучения;
- `notebooks/crm_intelligence_training.ipynb` — notebook для воспроизведения этапов обучения;
- `requirements.txt` — список библиотек;
- `app/` — Flask-приложение, UI и ML-логика.

## Запуск

1. Создать и активировать виртуальное окружение.
2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Запустить приложение:

```bash
python main.py
```

4. При необходимости инициализировать БД и сиды:

```bash
flask --app app init-db
flask --app app seed-users
flask --app app seed-clients
```

5. Открыть в браузере:

```text
http://127.0.0.1:5000/
```

## Датасет и модель

Встроенный датасет `data/crm_customer_churn.csv` подготовлен на основе открытого набора IBM Telco Customer Churn и приведён к CRM-ориентированным признакам:

- срок обслуживания;
- ежемесячная и накопленная выручка;
- количество подключённых сервисов;
- тип контракта;
- способ оплаты;
- интернет-услуга;
- семейный тариф;
- подключение техподдержки;
- целевой признак оттока.

Сохранённая модель `model.joblib` обеспечивает accuracy выше 0.77 на тестовой выборке.
